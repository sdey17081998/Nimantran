# Card Display Website

An Indian-themed card display website built with React and Vite.

## Adding Your Own Card Images

To add your own card images, follow these steps:

1. Navigate to the `public/images` folder in your project directory
2. Add your card images with the following exact filenames:

### Wedding Cards
- `wedding-1.jpg` - Traditional Hindu Wedding Card
- `wedding-2.jpg` - Royal Wedding Invitation
- `wedding-3.jpg` - Bengali Wedding Invitation

### Birthday Cards
- `birthday-1.jpg` - Birthday Celebration Card
- `birthday-2.jpg` - First Birthday Invitation
- `birthday-3.jpg` - Birthday Bash Card

### Rice Ceremony Cards
- `rice-ceremony-1.jpg` - Annaprashan Ceremony Card
- `rice-ceremony-2.jpg` - Mukhagni Ceremony Card
- `rice-ceremony-3.jpg` - Namkaran Ceremony Card

## Image Specifications
- Recommended format: JPG or PNG
- Recommended size: 400x300 pixels (4:3 aspect ratio)
- Maximum file size: 2MB per image

## Notes
- If an image fails to load, the website will automatically fall back to a gradient background with an emoji
- You can replace the images at any time without restarting the development server
- Make sure the filenames match exactly as shown above

## Running the Project

```bash
npm install
npm run dev
```

The website will be available at http://localhost:5173/
